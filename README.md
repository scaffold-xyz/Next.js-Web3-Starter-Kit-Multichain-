
This is the start of a comprehensive Web3 application built with modern technologies and multi-chain wallet connectivity.

## Tech Stack & Integrations

This application integrates the following technologies:

- **Next.js** - React framework with App Router for server-side rendering and routing
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Shadcn/ui** - Beautiful and accessible UI components built on Radix UI
- **TanStack Query** - Powerful data synchronization for React applications
- **Reown AppKit (WalletConnect)** - Universal wallet connection with support for all major chains
- **Multi-chain Support** - Integrated wallet connectivity across all blockchain networks

## Features

- 🔗 Universal wallet connection supporting all major blockchain networks
- 🎨 Modern, responsive UI with Tailwind CSS and Shadcn/ui components
- ⚡ Optimized data fetching and caching with TanStack Query
- 🌐 Server-side rendering with Next.js App Router
- 🔄 Real-time wallet state management
- 📱 Mobile-responsive design

## Getting Started

1. Go to [Reown Cloud](https://cloud.reown.com) and create a new project.
2. Copy your `Project ID`
3. Rename `.env.example` to `.env` and paste your `Project ID` as the value for `NEXT_PUBLIC_PROJECT_ID`
4. Install dependencies:
   ```bash
   npm install
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
